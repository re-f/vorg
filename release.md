# 0.0.7 版本变更
## 新增功能

- **ID 链接自动补全功能**：在输入 `[[` 后自动触发补全，显示所有可用的 ID 链接，支持模糊搜索和快速选择
- **工作区符号搜索功能**：使用 `Cmd+T` (Mac) 或 `Ctrl+T` (Windows/Linux) 快捷键搜索整个工作区中所有 .org 文件的标题，支持模糊搜索和快速跳转
- **Ctrl+C Ctrl+C 上下文操作功能**：支持 checkbox 状态切换（未完成 ↔ 完成 ↔ 部分完成）

## Bug 修复

- **改进 M-RET 对有序列表的支持和自动重新编号功能**：修复有序列表插入新项时的编号问题
- **修复预览滚动问题**：优化预览窗口的滚动同步
