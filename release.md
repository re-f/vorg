# 1.0.0 版本变更

## 新增功能

- **VOrg-QL 透视视图 (Perspectives)**：
  - 在侧边栏引入了全新的透视视图面板，支持快速切换常用的任务视图。
  - 支持自定义 VOrgQL 查询语句（如搜索特定优先级且未完成的任务）。
  - 支持保存自定义查询为永久透视视图。
- **全量工作区索引**：
  - 极大地提升了大型 Org 知识库的检索速度。
  - 工作区符号搜索 (`Cmd+T` / `Ctrl+T`) 现在支持拼音（首字母/全拼）匹配中文标题。
- **任务管理增强 (Emacs 标准对齐)**：
  - 完成任务时自动记录 `CLOSED` 时间戳，格式与 Emacs 完全一致。
  - 优化了优先级的循环切换和内部存储逻辑。
- **标签交互优化**：
  - 支持在标题行点击标签进行快速过滤。
  - 增强了标签输入时的自动补全体验。
- **Mermaid 图表渲染支持**：在预览中支持渲染 Mermaid 图表，包括流程图、时序图、类图、状态图、甘特图等多种图表类型。
  - 自动适配 VS Code 主题（亮色/暗色）
  - 导出 HTML 时保留图表渲染
  - 支持所有 Mermaid 官方图表类型  

## Bug 修复

- **跳转精度修复**：修复了在包含属性抽屉的复杂文档中，搜索跳转行号不准确的问题。
- **标签解析修复**：解决了重复标签导致的索引冲突问题，并支持了中文字符标签。
